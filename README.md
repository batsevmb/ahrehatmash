# Agragatmash landing page

Files:
- index.html
- styles.css
- script.js
- assets/ (logos, favicon, hero background)

How to run locally:
1) Open `index.html` in a browser
or
2) Serve with a simple static server:
   - Python: `python3 -m http.server 8080`
   - Node: `npx serve .`

Notes:
- No external libraries (fast load, works offline).
- Mobile menu + accordion are implemented in `script.js`.
